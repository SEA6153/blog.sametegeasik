Yazılım Yolculuğum

Ben Samet Ege Aşık. Hacettepe Üniversitesi Jeoloji Mühendisliği (Hidrojeoloji) mezunuyum. Alan dışı bir bölümden yazılım dünyasına geçiş yaptım.

2023–2024 yılları arasında BilgeAdam Akademi'de Java eğitimi aldım ve ardından T.C. Millî Eğitim Bakanlığı YEĞİTEK bünyesinde Java Backend Developer olarak çalışmaya başladım.

Spring Boot ve Java ile mikroservis mimarisi geliştirme konusunda derinleşiyorum. Özellikle servisler arası güvenli iletişim için OAuth2 tabanlı kimlik doğrulama yapıları üzerine çalışıyor, sistem performansını artırmak adına Redis ve Kafka entegrasyonlarını aktif olarak kullanıyorum. Sürekli teslimat (CI/CD) süreçleriyle yazılımı sadece geliştirmek değil, sürdürülebilir ve izlenebilir kılmak da ilgi alanlarım arasında. Öğrenmeyi, paylaşmayı ve üretmeyi odağıma alan bir geliştiriciyim.
